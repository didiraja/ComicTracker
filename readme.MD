# Minimal CRUD

### *"It creates, it reads, updates and deletes data??? OMG, its a complete CRUD!!!"*

### Api
  - **Server**
    - Hosted on [Render.com](https://render.com/)
    - Node
    - Express
    - Typescript
    - KISS architecture
    - Queries parametrization

  - **Database**
    - Hosted on [Supabase](https://supabase.com/)
    - PostgreSQL
    - Prisma ORM

### Front
  - Vite 
  - React
  - Typescript
  - Sass
  - Tailwind
  - Heroicons
